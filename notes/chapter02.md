## 2.1 变量定义

### 什么是变量？
变量是程序中用于**存储数据**的“容器”，在程序运行过程中值可以发生改变。

### Python变量的特点
- **弱类型语言**：变量的类型由其值的类型决定
- **无需预先定义**：直接赋值即可创建变量
- **动态类型**：同一变量可以赋不同类型的值

```python
# 变量定义示例
test = 'Hello World!'  # 字符串型变量
print(test)            # 输出 "Hello World!"

test = 123             # 整型变量（自动改变类型）
print(test)            # 输出 123

test = 10.5            # 浮点型变量
print(test)            # 输出 10.5

### Python 关键字详解

#### 逻辑值
- `True` - 真
- `False` - 假
- `None` - 空值

#### 条件判断
- `if` - 如果
- `elif` - 否则如果
- `else` - 否则

#### 循环控制
- `for` - 用于循环
- `while` - 当...时循环
- `break` - 跳出循环
- `continue` - 继续下一次循环

#### 函数和类
- `def` - 定义函数
- `return` - 返回函数值
- `lambda` - 定义匿名函数
- `class` - 定义类

#### 异常处理
- `try` - 尝试执行
- `except` - 捕获异常
- `finally` - 无论是否异常都执行
- `raise` - 抛出异常
- `assert` - 断言

#### 模块
- `import` - 导入模块
- `from` - 从...导入
- `as` - 起别名

#### 变量作用域
- `global` - 声明全局变量
- `nonlocal` - 声明外层变量
- `del` - 删除变量

#### 逻辑运算
- `and` - 与
- `or` - 或
- `not` - 非
- `is` - 判断是否是同一个对象
- `in` - 判断是否在序列中

#### 其他
- `pass` - 占位符，什么都不做
- `with` - 上下文管理
- `yield` - 生成器
- `async` - 异步函数
- `await` - 等待异步结果

# 同时定义多个变量
# 语法：变量1, 变量2, ... = 值1, 值2, ...
name, age = '张三', 18
print(name)  # 张三
print(age)   # 18

# 交换两个变量的值
# 方法1：使用临时变量
x, y = 5, 10
tmp = x
x = y
y = tmp

# 方法2：Python特有写法（推荐）
x, y = 5, 10
x, y = y, x  # 一行代码完成交换
print(x, y)  # 10 5
